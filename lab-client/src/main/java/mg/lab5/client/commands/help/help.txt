help : вывести справку по доступным командам
info : вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
show : вывести в стандартный поток вывода все элементы коллекции в строковом представлении
add {имя элемента} {длина маршрута} : добавить новый элемент в коллекцию
update id {имя элемента} {длина маршрута} : обновить значение элемента коллекции, id которого равен заданному
remove_by_id id : удалить элемент из коллекции по его id
clear : очистить коллекцию
save : сохранить коллекцию в файл
execute_script file_name : считать и исполнить скрипт из указанного файла. В скрипте содержатся команды в таком же виде, в котором их вводит пользователь в интерактивном режиме.
exit : завершить программу (без сохранения в файл)
remove_head : вывести первый элемент коллекции и удалить его
add_if_min {имя элемента} {длина маршрута} : добавить новый элемент в коллекцию, если его значение меньше, чем у наименьшего элемента этой коллекции
remove_lower {имя элемента} {длина маршрута} : удалить из коллекции все элементы, меньшие, чем заданный
remove_any_by_distance distance : удалить из коллекции один элемент, значение поля distance которого эквивалентно заданному
filter_by_distance distance : вывести элементы, значение поля distance которых равно заданному
filter_greater_than_distance distance : вывести элементы, значение поля distance которых больше заданного
